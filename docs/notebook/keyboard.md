# Keyboard Shortcuts

Press **?** anywhere in the notebook (outside a code editor) to show the shortcuts modal.

| Shortcut | Action |
|----------|--------|
| `Shift + Enter` | Run the current cell |
| `?` | Show keyboard shortcuts |
| `Esc` | Close modal |
| `Ctrl + Z` | Undo (in editor) |
| `Ctrl + Shift + Z` | Redo (in editor) |
